## JavaScript

A linguagem de programação interpretada, originalmente desenvolvida para permitir a interação de scripts com o usuário sem passagem pelo servidor, é bastante utilizada também em desenvolvimento web (junto ao HTML, o CSS e ao PHP), criação de jogos e aplicativos para celular.
A linguagem foi criada por Brendan Eich em 1995, e sua popularidade é decorrente do fato de ela ser uma linguagem extremamente flexível e multifacetada. Suas principais funcionalidades e aplicações são para adicionar comportamento interativo em páginas web, criar páginas na web e em aplicativos, construir web servers e desenvolver aplicações e desenvolvimento de jogos.
Entre as maiores vantagens para utilização do JavaScript é por ser uma linguagem de programação simples, universal e multiplataforma, rápido desenvolvimento e comunicação assíncrona.

## Python

A linguagem de programação Python foi criada em 1991 por Guido Van Rossumem e apesar de simples, é uma linguagem muito poderosa, que pode ser usada para desenvolver e administrar grandes sistemas [1].
Uma das principais características que diferencia a linguagem Python das outras é a legibilidade dos programas escritos, onde o uso de marcações é reduzido, deixando a linguagem visualmente mais limpa, de fácil compreensão e leitura. 
Atualmente, a linguagem é gerenciada pela Python Software Foundation. Possui uma licença de código aberto, chamada Python Software Foundation License, que é compatível com a GNU General Public License a partir da versão 2.1.1 e incompatível em certas versões anteriores.É uma linguagem de programação multi-paradigma, pois suporta orientação de objeto, programação imperativa e, em menor escala, programação funcional. 
As vantagens para sua implementação são as de que o Python possui Grande quantidade de bibliotecas disponíveis, grande comunidade de usuários, documentação extensa e número expressivo de oportunidades no mercado de trabalho.
## C++

A Linguagem C++ é uma extensão da linguagem C que incorpora recursos de Orientação a Objetos que tinha como objetivo o desenvolvimento desta linguagem para melhorar uma versão do núcleo Unix. A linguagem C++ herdou todas as características de C, e adiciona recursos de programação orientada a objetos [2]. Bjarne Stroustrup desenvolveu o C++ em 1983 no Bell Labs, adicionando–o novas características, como funções virtuais, sobrecarga de operadores, herança múltipla, gabaritos e tratamento de exceções.
No que se diz respeito as principais vantagens de sua utilização estão por ser uma linguagem que suporta múltiplos paradigmas, liberdade para o programador escolher as opções, mesmo sendo a opção erradas e muitos códigos podem ser transferidos para C facilmente, pois o C++ foi criado para ter compatibilidade com o C.

#PHP

PHP é uma linguagem para criação de scripts para a Web do lado servidores embutidos em HTML, cujo código-fonte é aberto, que é compatível com os mais importantes serviços Web. O PHP permite incorporar fragmentos de códigos em páginas de HTML normais - códigos esses que são interpretados à medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem de “cola”, facilitando a conexão de suas páginas Web com o banco de dados do lado do servidor [3].
Entre suas maiores vantagens podemos citar facilidade no aprendizado, pois para construir essa linguagem são necessários alguns elementos de outras linguagens; linguagem open source, então não possui custos e com melhorias constantes, pois qualquer programador pode melhorar seu código, retirando possíveis bugs ou adicionando mais funções; compatível aos mais diversos bando de dados, adicionado importância no dinamismo em carregar os elementos das páginas sem falhas e com agilidade.

## Java

Projetada pelo Sun Microsystems, a linguagem de programação Java é uma das mais populares do mundo, com um vasto repertório de profissionais, aplicações, documentação e comunidade. O pensamento inicial, em 1991, era bem similar ao que o Internet of Things (IoT) busca entregar: permitir que os dispositivos eletrônicos pudessem se comunicar entre si [4]. 
Em 2021, a linguagem de programação celebrou o aniversário de 26 anos. Mesmo nessa idade, ela continua amplamente utilizada para soluções web, desktop e mobile por desenvolvedores back-end, big data e Android, e diversas outras aplicações. Ela pode não ser a linguagem de programação número um no momento, mas definitivamente está entre as mais populares.
Para ampliar melhor as características da linguagem JAVA, é importante ressaltar sobre os tipos de dados que são trabalhados. Java é uma linguagem fortemente tipada, ou seja, todas as variáveis precisam ter um tipo. Os tipos podem ser primitivos ou não-primitivos [5]. 
a) Primitivo: define oito tipos de dados, com o respectivo tamanho da memória necessária para armazená-los através da tabela de alocação, sem possuir métodos adicionais. São dados primitivos: 
i) byte, com tamanho de 1 byte = 8 bits, usado para armazenar número inteiros de -128 a 127. 
ii) short, com tamanho de 2 bytes = 16 bits, usado para armazenar números inteiros de -32.768 a 32.767.
 iii) int, com tamanho de 4 bytes = 32 bits, usado para armazenar números inteiros de -2.147.483.648 a 2.147.483.647.
 iv) long, com tamanho de 8 bytes = 64 bits, usado para armazenar números inteiros de -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807. 
v) float, com tamanho de 4 bytes = 32 bits, usado para armazenar números racionais com tamanho suficiente de 6 e 7 dígitos decimais. 
vi) double, com tamanho de 8 bytes = 64 bits, usado para armazenar números racionais com tamanho suficiente de 15 dígitos decimais. 
vii) boolean, com tamanho de 1 bit, usado para armazenar dois valores, verdadeiro e falso
viii) char, com tamanho de 2 bytes, usado para armazenar um único caractere, letra, ou valor ASCII. Alguns dos tipos trata apenas se número, possuindo uma subclassificação em: • Tipos inteiros: como o próprio nome define. • Tipos flutuantes: números racionais com um ou mais dígitos decimais
 b) Não-primitivos: String, Array e outras classes (objetos). 
Os tipos não-primitivos são referenciados a objetos e são definidos pelo desenvolvedor, não pelas bibliotecas Java. Assim, eles permitem a invocação de métodos, podendo armazenar valores ou simplesmente "null", e consomem um espaço na memória de tamanho fixo. Todos os objetos usam alocação heap-dinâmica e precisam do uso do garbage collector para a limpeza da memória. É boa prática de programação Java iniciar uma variável não-primitiva com letra maiúscula.
Em uma visão geral, as vantagens de Java se encontram em sua simplicidade, pois apresenta sintaxe direta e acolhedora, sendo fácil de escrever, aprender, depurado facilmente, consequência de: 
i) a orientação a objetos aumenta bastante a flexibilidade e a capacidade de reutilização do código com as implementações dos método 
ii) os modificadores e encadeamentos aumenta a segurança e a vinculação de dados, ao mesmo tempo que gerencia o acesso para as classes;
 iii) é uma linguagem barata, pois podem ser facilmente executadas em qualquer máquina; 
iv) como se trata de uma linguagem distribuída, permite o compartilhamento de dados e programas entre vários computadores, melhorando o desempenho; 
v) Como Java trabalha com multi thread, permite obter o máximo de utilização da CPU. 
Várias threads compartilham uma área de memória comum e aumenta e eficiência o desempenho do aplicativo;
 vi) a JVM faz com que sempre que os objetos não são mais usados pelo programa, e não se referem a nada que não precise ser desreferenciado ou removido pela programação explícita; 
vii) java pode ser executado em diferentes plataformas, fornecendo um forte vantagem em portabilidade; 
viii) é uma linguagem de programação de alto nível, pois é uma linguagem legível por humanos. É semelhante à linguagem humana e tem uma sintaxe muito simples e fácil de manter
Em contrapartida, como se trata de uma linguagem usada a muitos anos, desde sua criação, foram feitas diversas atualizações que, mesmo mantendo os critérios estáveis, não solucionaram totalmente os problemas. Desta foram, o Java também apresentam algumas desvantagens: 
i) oferece baixo desempenho comparados a linguagens como C e C++, pois cada código deve ser interpretado para o código de nível de máquina.
 ii) a linguagem somente se concentra no armazenamento, porém peca no backup;
 iii) requer quantidade maior de espaço de memória em comparação com outras linguagens como C e C ++. Durante a execução da coleta de lixo, a eficiência da memória e o desempenho do sistema podem ser afetados adversamente.
 iv) códigos para implementação possuem muitas frases longas e complexas, que são difíceis de entender a curto prazo.
